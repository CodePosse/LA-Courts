Site Builder Specifications

Overview
The Site Builder is a core feature that allows users to construct and manage the navigational and content structure of their websites. Each website corresponds to a unique subdomain on the platform. The Site Builder enables users to control page structure, design, visibility, and publication status with an easy-to-use drag-and-drop interface.

Objective
The Site Builder aims to provide users with a streamlined interface to manage their website’s structure, layout, and content. It will allow for full control over navigation, page organization, visibility, and future dynamic content integration.

Website Navigation Defaults
The following default pages are generated when the website is created:

  • Home (User-defined)
  • Programs
  • Jobs
  • Give (Donations)
  • About Us
  • News
  • Gallery (Not MVP)
  • Contact Us
These default pages cannot be deleted, but can be edited or hidden. Hiding does not affect dynamically generated content.

3. Navigation Settings Users can:

  • Choose whether a page appears in the navigational header.
  • Set visibility rules for each page (public, logged-in users only, admin only).

4. Page Design

  • Users design individual pages using an embedded BeeFree template designer.
  • Users can preview pages within the BeeFree interface before publishing.

5. Publish/Unpublish Workflow

  • Pages can be saved as drafts or published.
  • Users can unpublish pages at any time, removing them from public view. The page’s status should reflect whether it’s published or unpublished.
  • All actions should trigger a confirmation alert, and an alert should be shown when publishing or unpublishing.
  • When a page is published, it should be displayed as "Published" with an option to unpublish.

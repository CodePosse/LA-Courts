Creative Brief:

This is a user flow demo. the content herein is purely for suggestion and is not to be considered for production. This excersize is to get a look and flow, not to make a website, that conent will be edited and approved by others.

Not all the elements are heare, we focused on the prime real estate of the page by decluttering it. Anything that needs to return should only be done below what we see here. Eliminating the overload of the text from the previous site makes this userflow far more efficient for people that are not enthusiastic about jury service to get them easily processed and assigned.

The rates of completion should increase; with FTA, customer support calls, and people igoring service will all decrease.

User Flow
Steps and States

  1. Step 1: Data Collection
    1. 1.1 All items incomplete/unverified
    2. 1.2 Some items incomplete/unverified
    3. 1.3 All items completed/verified
    4. Personal Info Form
    5. Eligibility Questionnaire
  2. Step 2: Pending
    1. 2.1 Educate, Set Reminders
    2. 2.2 Change Requested
    3. 2.3 Change Resolution
  3. Step 3: Service Week
    1. 3.1 On Call
    2. 3.2 Assigned
  4. Step 4: End States
    1. 4.1 Completed
    2. 4.2 Excused
    3. 4.3 Service Week Passed
    4. 4.4 Failure To Appear
    5. 4.5 Ineligible

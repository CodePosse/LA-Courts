Making includes without server-side code.

As you may know, having code on one site appear on multiple external is not always possible using the include schema. By using one JS file, you can publish content across multiple sites.

Let's start with a blank text file, call it "content.js". It will hold your common content across multiple sites.

Inside the file we will use the document.write method. The code is very simple and looks like this: document.write('hello world!'). Each item you add must be on ONE LINE, so turn wordwrap off in your editor.

To prevent it from breaking when you add content containing quote " marks, you need to use the backslash technique. "\

document.write('URL Link | plain link')

Download the sample: common.js

Now that you have added all this content to your file, remember that unless you put in p or br or other formatting it will just wind up in the exact spot you place it.

Adding to a page once you have written your JS is VERY easy, just plug this in wherever you want it to appear.
